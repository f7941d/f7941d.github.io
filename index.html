Additional information:

IP and Owner are removed from the device info, because it is considered sensitive information.

⚠️ (Password Unknown/Forgotten) - when used netsh wlan show profile key="clear" on the network, no password was returned (the network was forgotten)

✅ (Password Verified) - one of the passwords (the underlined one) is the correct network password

❌ (Password Unverified) - none of the passwords are correct


How it works:

  1. netsh wlan show profiles is ran
  2. all known networks are extracted
  3. netsh wlan show profile name="{name}" key=clear is ran for each of the extracted networks, key=clear makes the password plain text
  4. The password is extracted from each one, and saved into a variable
  5. curl ifcfg.me is ran, to get the public IP of the device (not used)
  6. systeminfo is ran, to get the System Manufacturer, Model, OS Name, Host Name, and Owner of the device
  7. Everything is saved into variable with the json syntax
  8. All info is sent using the Telegram Bot Get API
  9. The received message is uploaded to GitHub, where the website automatically updates
